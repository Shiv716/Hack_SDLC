# Hack_SDLC

- Run backend: ctxcli.py

```
python ctxcli.py
```

- Run commands

```
python ctxcli.py --help
python ctxcli.py show-graph
python ctxcli.py validate
python ctxcli.py compliance
python ctxcli.py ci --trigger
python ctxcli.py openweb <URL here>
```
